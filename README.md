# FUTURE_PE_01 – AI Website Copy Generator

## 📌 Task Overview
This project is part of the Prompt Engineering (PE) track.  
The objective was to generate conversion-focused website copy for a local business using structured AI prompts.

For this task, I created complete homepage and service page content for a fictional fitness brand, **FitLife Studio (Bangalore)**, targeting working professionals and college students.

---

## 🎯 Project Objective
- Generate persuasive homepage content
- Create detailed service page content
- Structure pricing, FAQs, and CTAs
- Optimize content for engagement and conversions
- Document prompts and outputs professionally

---

## 🛠 Tools Used
- ChatGPT (Content Generation)
- Google Docs (Formatting & Review)
- GitHub (Documentation & Submission)
